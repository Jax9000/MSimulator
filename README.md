# MSimulator

Objective programming example in c++
typeinfo, template, virtual function, 


input

X Y t

T1 x1 y1 vx1 vy1 r1 m1
X Y - size of area
t - iterations count
T - type (normal N, dual B, dual-fissile, fissile, photonic)
x - x position
vx - x speed
vy - y speed
r - radius (0-0.5)
m1 - mass (photonic has no mass)


example:
10 15 30000
N1 8 6 1 0.5 0.3 0.4
P2 3 -2 -0.3 1 0.4
